## 🛠️ Core Features
* **Strategy 1 PDF Parsing:** Automatically shreds heavy documents down to discrete page targets.
* **Vector Canvas HUD:** Draw force vectors, highlights, and custom physics annotations directly onto question images.
* **Security Protocol:** Local compiled protection via hardware binding alongside disabled browser inspector overrides.
