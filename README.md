\# Tactical Study Platform \& Skill Enhancer



A localized, fully offline interactive platform to manage, view, flag, and annotate study images and questions. Designed specifically for Windows.



\---



\## 🚀 How to Set Up and Run (First Time User)



Follow these simple steps to get the platform working on your laptop:



\### Step 1: Install Python

If you do not have Python on your laptop yet:

1\. Go to \*\*python.org\*\* and download the official installer for Windows.

2\. Run the installer.

3\. \*\*CRITICAL STEP:\*\* At the very bottom of the first setup screen, look for a checkbox that says \*\*"Add python.exe to PATH"\*\*. You \*\*MUST\*\* check this box before clicking install.



\### Step 2: Download this Project

1\. Download this entire folder from GitHub as a ZIP file (Click the green \*\*Code\*\* button at the top right -> \*\*Download ZIP\*\*).

2\. Extract the ZIP file anywhere on your laptop (e.g., your Desktop or E: Drive).



\### Step 3: Run the Platform

1\. Double-click the file named \*\*`Initiate\_Study\_Work.bat`\*\*.

2\. A black terminal window will open. If it is your first time running it, it will automatically download the required interface files securely in the background.

3\. A box will pop up asking you to select your study folder. Choose the main folder on your computer where your subject directories and images are located.

4\. The platform will automatically open inside your web browser. You are ready to study!



\---



\## 🎨 Core Features

\* \*\*Fully Offline:\*\* Works seamlessly even without an active internet connection.

\* \*\*Vector \& Biomechanics Canvas:\*\* Draw force arrows, kinematic arcs, or write hints directly over question images. Everything auto-saves.

\* \*\*Interactive Folder Tree:\*\* Browse your structured notes and study question images via a sleek sidebar.

\* \*\*Gamified Metrics:\*\* Track your study streaks and level up as you complete tasks.
